# NEONE — Frontend (React + Vite + Tailwind)

## Ishga tushirish
```bash
npm install
npm run dev
```

Brauzerda `http://localhost:5173` manzilini oching.

## Loyiha tuzilishi
- `src/App.jsx` — sahifalar orasida navigatsiya (Bosh sahifa / Kategoriya / Mahsulot)
- `src/pages/Home.jsx` — bosh sahifa (hero, kategoriyalar, trend mahsulotlar, NEONE Club banner)
- `src/pages/Category.jsx` — kategoriya sahifasi (yon panel, filtr, saralash, mahsulotlar grid)
- `src/pages/Product.jsx` — mahsulot sahifasi (galereya, ranglar/o'lchamlar, tavsif, sharhlar, o'xshash mahsulotlar)
- `src/components/` — Header, Footer, ProductCard, SearchOverlay (qidiruv mega-menyu)
- `src/data/products.js` — namunaviy mahsulot va kategoriya ma'lumotlari

## Build
```bash
npm run build
```
